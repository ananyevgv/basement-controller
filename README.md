

# basement Water Controller ESPHOME Версия 3.0

ESPHome-прошивка для умного контроллера управления водой и мониторинга в подвале с цветным TFT-дисплеем на базе платы **KinCony KC868-A6**.

---

## 📋 О проекте

Проект представляет собой полнофункциональную систему автоматизации подвала, построенную на базе промышленного контроллера KinCony KC868-A6. Система обеспечивает:

- **Обнаружение протечек** воды с автоматическим перекрытием
- **Управление краном** (ручное и автоматическое)
- **Мониторинг насоса** (ток, время работы, счетчик включений)
- **Контроль качества воздуха** (температура, влажность, давление, IAQ, CO₂, VOC)
- **Учет потребления воды** через счетчик
- **Визуализацию всех параметров** на TFT-дисплее
- **Управление 4 нагрузками** (освещение, розетки и т.д.)
- **Интеграцию с Home Assistant**

---

## 🎯 Ключевые возможности

### 💧 Управление водой
- Автоматическое перекрытие крана при обнаружении протечки (2 независимых датчика)
- Ручное управление краном через кнопку или интерфейс
- Датчики конечных положений для точного контроля состояния
- Визуальное отображение состояния крана на дисплее

### 📊 Мониторинг
| Параметр | Способ измерения |
|----------|------------------|
| Протечка | 2 проводных датчика UJIN  |
| Ток насоса | Датчик тока ZMCT103C WC |
| Время работы насоса | Автоматический подсчет |
| Счетчик включений насоса | Автоматический подсчет |
| Расход воды | BLE счетчик Элехант |
| Температура воздуха | BME688 (I2C) |
| Влажность воздуха | BME688 (I2C) |
| Давление атмосферное | BME688 (I2C) |
| Качество воздуха (IAQ) | BME688 (BSEC) |
| CO₂ / VOC | BME688 (BSEC) |
| Температура воды | XDB401 (I2C) |
| Давление воды | XDB401 (I2C) |
| Заряд батареи счетчика | BLE счетчик Элехант |

### 🔌 Управление нагрузками
- 4 реле (освещение)
- Сценарии группового включения/выключения
- Кнопочное управление с поддержкой мультикликов:
  - Короткое нажатие → поочередное включение реле 1→2→3→4
  - Двойное нажатие → включить все реле
  - Длинное нажатие → выключить все реле

### 📟 Визуализация на дисплее
Цветной TFT-дисплей 1.77" ST7789V (170×320) отображает:

- Статус Wi-Fi (графический индикатор)
- Состояние крана (открыт/закрыт)
- Датчики протечки (визуальная сигнализация)
- Показания счетчика воды (м³)
- Температуру и давление воды
- Статус насоса (вкл/выкл) + счетчик включений
- Заряд батареи счетчика
- Параметры ИБП (APC): напряжение сети, заряд АКБ
- Качество воздуха: температура (с цветовой индикацией), влажность, давление (с прогнозом погоды), IAQ
- Состояние 4-х реле

### 🔊 Звуковое оповещение
- Сигнализация при обнаружении протечки
- Воспроизведение мелодий в формате RTTTL

## 🛠️ Аппаратная платформа

Проект реализован на контроллере **KinCony KC868-A6**:

| Компонент | Спецификация |
|-----------|--------------|
| **Микроконтроллер** | ESP32-WROOM-32 (240 МГц, 4 МБ Flash) |
| **Реле** | 6 реле 10A 220V  |
| **Дискретные входы** | 6 оптоизолированных (для датчиков) |
| **Аналоговые входы** | 4 входа 0-5 В (для датчиков протечки и тока) |
| **Интерфейсы** | RS232 (DB9), RS485, I2C, SPI, 1-Wire |
| **RTC** | DS1307 с батарейным питанием |
| **USB** | Type-C (CH340C) для программирования |
| **Питание** | 12 В DC |

## 🚀 Установка и настройка

### 1. Требования
- ESPHome ≥ 2023.12.0
- Плата KinCony KC868-A6
- USB-кабель Type-C


#### Подключение датчиков
| Датчик | Порт/пин | Примечание |
|--------|----------|------------|
| BME688 | I2C (GPIO4, GPIO15) | Адрес 0x77 |
| XDB401 | I2C (GPIO4, GPIO15) | Адрес 0x7F |
| Протечка 1 | GPIO32 (1-Wire) | Удалить R23, подтянуть резистором 100Ω к GND |
| Протечка 2 | GPIO33 (1-Wire) | Удалить R31, подтянуть резистором 100Ω к GND |
| Ток насоса | GPIO36 (ADC) | Датчик тока ZMCT103C |
| Кнопка реле | GPIO12 | INPUT_PULLDOWN |
| Кнопка крана | GPIO13 | INPUT_PULLUP |
| RS485 | GPIO27 (TX), GPIO14 (RX) | Для связи с ИБП |
| PCF8574 (входы) | I2C (GPIO4, GPIO15) | Адрес 0x22 |
| PCF8574 (выходы) | I2C (GPIO4, GPIO15) | Адрес 0x24 |
| Дисплей ST7789V | SPI (GPIO18, GPIO19, GPIO23, GPIO5) | 5V питание |
